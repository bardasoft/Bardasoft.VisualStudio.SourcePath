# Checklist antes de publicar

- [ ] El VSIX compila correctamente en Release.
- [ ] El VSIX se instala y funciona localmente.
- [ ] `source.extension.vsixmanifest` tiene el nombre y versión correctos.
- [ ] `marketplace/vs-publish.json` contiene tu Publisher ID real.
- [ ] `identity.internalName` está correcto y será permanente.
- [ ] `marketplace/overview.md` está revisado.
- [ ] El enlace de PayPal fue reemplazado.
- [ ] La URL del repositorio GitHub es correcta.
- [ ] Las imágenes de `marketplace/images` se ven bien.
- [ ] No hay tokens ni secretos dentro del repositorio.
