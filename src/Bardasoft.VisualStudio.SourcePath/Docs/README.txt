BARDASOFT SOURCEPATH

Bardasoft SourcePath is a free and open-source Visual Studio extension.

WHAT IT DOES
- Shows the full source file path of the active document.
- Displays the path in a discreet footer inside the Visual Studio editor.
- Helps identify the exact file being edited in large solutions.

AVAILABLE ACTIONS
- Right-click the footer to copy the full path, file name, or folder path.
- Ctrl+Click the footer to open the file location in Windows Explorer.
